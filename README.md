# Requirements

You will need (Deno)[https://deno.land] Installed

# Config

Copy the following to a `config.ts` file to the repository root and fill in the values.

```ts
export default {
  SLACK_CLIENT_ID: "", // not used yet
  SLACK_CLIENT_SECRET: "", // not used yet
  DISCORD_BOT_TOKEN: "",
  HTTP_PORT: 3333,
  LOG_LEVEL: 'INFO',
};
```
